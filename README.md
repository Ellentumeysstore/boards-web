blimp-boards-web
================

Blimp Boards Front-End for the web client.

Dependencies
------------

    $ npm install -g brunch gulp

Install
-------

    $ git clone git@github.com:GetBlimp/boards-web.git
    $ cd boards-web
    $ npm install
    $ bower install
    
Run
---

    $ brunch w -s
    
Tests
-----
    
    $ gulp test
